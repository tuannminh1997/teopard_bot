V6.1 - Output wording cleanup for MACD histogram

Fixed:
- Removed `Hist` / `MACD_hist` wording from the analysis data shown to the LLM.
- Added `macd_momentum_text()` so MACD histogram is described as Vietnamese wording:
  - `động lượng MACD âm ...`
  - `động lượng MACD dương ...`
- Updated `analyze_system_prompt.txt` to forbid user-facing words: Hist, MACD_hist, Histogram, histogram.
- Updated `sanitize_user_output()` to clean accidental model output such as:
  - `Hist -75.84` -> `động lượng MACD -75.84`
  - `MACD_hist=-75.84` -> `động lượng MACD=-75.84`
- Updated `summarize_reasoning()` to avoid and sanitize Hist/MACD_hist too.

Meaning:
- Hist = MACD histogram = MACD line minus Signal line.
- Negative value means bearish/down momentum is stronger.
- Positive value means bullish/up momentum is stronger.
